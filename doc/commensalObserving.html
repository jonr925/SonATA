Commensal Observing

An observer at the ATA may wish to gather data using the correlator at the same time SonATA is looking for signals. This is called Commensal Observing. When commensal observing is in effect the ATA correlator data gathering requires that the telescope occasionally point to a calibrator for one or more minute.

To falicitate commensal observing the Seeker has several parameters that can control how often the telescope is pointed to a calibrator, and how long to stay pointed at the calibrator.

These script commands control the commensal observing:

  • To set comensal observing on/off:
    • sched set comcal <on|off> (off is the default>
  • To Specify how long between observing a calibrator:
    • sched set comcalinterval <minutes> (default 60 minutes, minimum 30, max 360)
  • To specify how long the observation should be:
    • sched set comcallength <minutes> (default 2 minutes, minimum 1, max 20)

It is important to note that the SonATABackendServer is continually broadcasting the telescope pointing location to 227.24.24.2:24243, over UDP for the correlator commensal observation side.
