FFTW Installation

Go to http://www.fftw.org/download.html

Download FFTW 3.2.1 as /tmp/fftw-3.2.1.tar.gz

   cd   /usr/local (as root)

   tar   -zxpf   /tmp/fftw-3.2.1.tar.gz

    cd     fftw-3.2.1

    ./configure     --enable-threads     --enable-float     --disable-fortran

    make

    make     check

    make     install
