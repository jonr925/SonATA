Prelude Web Control Interface User's Guide

Overview: 

The Prelude SETI observing system can be controlled and
monitored by use of a web server application running on
port 8080 of the primary observing host.
The access URL is:

   http://[prelude host]:/8080

Currently the primary host is sse3, so this would become

   http://sse3:/8080

The web interface allows for system startup and shutdown,
status displays of currently running observations (aka
activities), displays of observing data products as
waterfalls and baseline plots, database queries of activity
history and signal detections, and so on.

The web site pages are outlined and described below.

Note:
You'll need to have javascript enabled on your browser to
make use of most of the features described.

Site Map:

  1. Main Menu Bar Links (appear at top of all pages)
    • Home (Main Page)
    • Status
    • StatusDetail
    • Candidates
    • ObsHist
    • Logs
    • Archive
    • DbQueries
    • PointMap
    • SeekerCmd
    • SysCntl

  2. Home (Main) Page
    • Main Menu Bar
    • Links to other information
      • Prelude docs
      • Observing aids (JPL Horizons, spacecraft targets, visibility charts etc.)
      • Hat Creek site information

  3. Status (summary, auto-refresh)
    • Currently running activities
    • Recently observed targets
    • Recent observation history
    • System log tail
    • Error log tail

  4. Status Detail
    • Running activities
    • Individual component status

  5. Candidates
    • Table of candidates from last 50 activities (freq, power, drift, etc).

  6. ObsHist (observation history)
    • Last 50 activities (Act #, signal count totals, etc).

  7. Logs (current version; auto-refresh)
    • System | Error | Debug | Components

  8. Archive (waterfalls, baselines, activity summaries)
    • daily data products in YYYY-MM-DD directories
    • confirmation data (most recent activity)
    • all logs (system, error, templogs)

  9. DbQueries (database queries)
    • Form for selecting query type and parameters

  10. PointMap
    • RA/Dec map of current beam pointings

  11. SeekerCmd
    • Command extry form and response log
    • System & error log tails

  12. SysCntl
    • System startup & shutdown command forms 
Misc:
- runs under tomcat 
- auto page refreshes, which pages, turning refresh on & off
- status indicators (green & red bars) etc.
- allow time for startup & shutdown
